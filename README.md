# How to deploy

1 `git clone https://github.com/Mr-ShiRoNo/Starry-Sky-Source.git`

2 `cd Starry-Sky-Source`

3 `pip3 install -r requirements.txt`

(If there are any missing modules, please add them separately.)

4 `python3 Starry-Sky.py`
